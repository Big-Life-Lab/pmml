# pmml

R Package to convert a set of 
[model parameters](https://github.com/Big-Life-Lab/model-parameters) files to 
PMML.

## Repo Organization

**[R](./R)**: Contains all the R code for the project
**[tests](./tests)**: Contains all the tests for the project

## Working with the Files

Each folder in the repo contains a README file that goes over the files within
that folder. The README files also contains instructions and rules for editing
and working with the files in that folder that should be followed.

## Common Commands

* Run the tests, `rig run -r 4.1.2 -e "devtools::test()"`
